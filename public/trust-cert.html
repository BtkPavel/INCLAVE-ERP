Установка сертификата на iPhone

Safari не загружает сайт полностью, пока не доверяет нашему сертификату. Сделайте один раз:

  1. Нажмите кнопку ниже и разрешите установку профиля.
  2. Откройте Настройки → вверху появится Профиль загружен → Установить.
  3. Настройки → Основные → Об этом устройстве → Доверие сертификатам → включите доверие для INCLAVE ERP.
  4. Откройте https://87.199.192.18/ — предупреждение исчезнет.
  5. Добавьте приложение на экран «Домой» заново.
